# Week 5-6 課題：React + DB連携 TODOアプリ

## 概要

Week 3-4で作成したReact + TypeScript TODOアプリを拡張し、Firebaseと連携させる。
認証機能を追加し、**ユーザーごとにTODOデータを管理**できるようにする。

## 技術スタック

- **フロントエンド**: React + TypeScript (Vite)
- **バックエンド/DB**: Firebase (Firestore)
- **認証**: Firebase Auth
- **スタイリング**: Sass (SCSS)

### 2. TODOダッシュボード (`/` or `/dashboard`)

#### 2.1 統計カード（ヘッダー部分）

以下の4つの統計情報をカード形式で表示：

| カード     | 内容                           | アイコン色  |
| ---------- | ------------------------------ | ----------- |
| 総タスク数 | 全TODOの件数                   | 青          |
| 進行中     | ステータスが「進行中」の件数   | 黄/オレンジ |
| 完了済み   | ステータスが「完了」の件数     | 緑          |
| 期限切れ   | 期限が過ぎた未完了タスクの件数 | 赤          |

#### 2.2 TODO追加フォーム

- [ ] タイトル入力フィールド（プレースホルダー: "TODOを入力..."）
- [ ] 期限選択（日付ピッカー）
  - **デフォルト値: 本日の日付**
- [ ] 優先度選択（ドロップダウン）
  - 高
  - **中（デフォルト）**
  - 低
- [ ] 「+ 追加」ボタン
- [ ] バリデーション（タイトル必須）

#### 2.3 フィルター機能

右サイドまたは上部にフィルターパネル：

- [ ] **ステータス**フィルター（ドロップダウン）
  - すべて
  - 進行中
  - 完了済み
- [ ] **優先度**フィルター（ドロップダウン）
  - すべて
  - 高
  - 中
  - 低
- [ ] **期限**フィルター（ドロップダウン）
  - すべて
  - 期限切れ
  - 今日
  - 今週

#### 2.4 TODOリスト表示

各TODOアイテムに以下を表示：

- [ ] 完了チェックボックス（左側）
- [ ] タイトル
- [ ] 優先度バッジ（例: `優先度: 高`）
  - 高: 赤系
  - 中: 黄系
  - 低: 青/グレー系
- [ ] ステータスバッジ（例: `進行中` / `完了済み`）
  - 進行中: 青系
  - 完了済み: 緑系
- [ ] 編集ボタン（ペンアイコン）
- [ ] 削除ボタン（ゴミ箱アイコン）

#### 2.5 TODO編集機能（インライン編集）

編集ボタンクリック時：

- [ ] タイトルが編集可能なテキストフィールドに変化
- [ ] 期限が編集可能に
- [ ] 優先度が編集可能（ドロップダウン）
- [ ] 「保存」ボタン
- [ ] 「キャンセル」ボタン

#### 2.6 TODO削除機能

- [ ] 削除ボタンクリックで削除
- [ ] 確認ダイアログ表示

#### 2.7 TODO完了切り替え

- [ ] チェックボックスクリックで完了/未完了を切り替え
- [ ] 完了時はタイトルに打ち消し線（任意）
- [ ] ステータスバッジが「完了済み」に変化
